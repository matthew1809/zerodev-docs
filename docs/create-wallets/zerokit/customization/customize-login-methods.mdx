# Customize login methods

You can configure which login methods to show by using the `enabledConnectors` prop.
```jsx
<ZeroKitProvider projectId="<your-project-id>" enabledConnectors={["google", "metaMask"]}>
  <YourApp />
</ZeroKitProvider>
```
