# Getting Started

### Step 1: Sign up
Fill out the [closed beta signup form](https://docs.google.com/forms/d/e/1FAIpQLSfip6bMIYvd2Z5LDr3IRt-kCEc6x38TybcDe6gk_TRKvT-fMw/viewform) to get a `projectId`.


### Step 2: Installation
<Tabs>
  <TabItem value="npm" label="npm" default>
    <pre>
        <codeblock>npm install --save @zerodevapp/zerokit</codeblock>
    </pre>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <pre>
        <codeblock>yarn add @zerodevapp/zerokit</codeblock>
    </pre>
  </TabItem>
</Tabs>

### Step 3: Setup the SDK

ZeroKit is built on top of RainbowKit, so the dev experience is almost the same.  [See here](/rainbow-kit) for convert a RainbowKit app to a ZeroKit app.

You will need your `projectId` here to setup the SDK.

```jsx
import '@zerodev/zerokit/styles.css';
import { ZeroKitProvider } from "@zerodevapp/zerokit";
```
```jsx
<ZeroKitProvider projectId="<your-project-id>">
    <YourApp />
</ZeroKitProvider>
```

### Step 4: Action

In `YourApp`, use the `ConnectButton` component to create a connect button, and use Wagmi to interact with the user's wallet.

```jsx
import { ConnectButton } from "@zerodevapp/zerokit";
```
```jsx live
<ConnectButton label="Social Wallet" />
```

[See here](/examples/sponsored-mint) for a full example.
