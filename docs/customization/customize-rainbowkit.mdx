# Customize RainbowKit
Furthermore, the `ZeroKitProvider` accept all `RainbotKitProvider` props, like:
```jsx
<ZeroKitProvider projectId="<your-project-id>" showRecentTransactions>
  <YourApp />
</ZeroKitProvider>
```
