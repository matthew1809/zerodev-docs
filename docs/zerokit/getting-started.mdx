# Getting Started

ZeroKit is a turn-key AA wallet widget that can be embedded inside your app with just 5 lines of code.  Once embedded, your users can sign in with social account and Web3 wallets, and then interact with your app through an AA wallet, enabling all kinds of [powerful features](/category/account-abstraction).

### Step 1: Sign up

Fill out the [beta signup form](https://docs.google.com/forms/d/e/1FAIpQLSfip6bMIYvd2Z5LDr3IRt-kCEc6x38TybcDe6gk_TRKvT-fMw/viewform) to get a `projectId`.

### Step 3: Setup the SDK

ZeroKit is built on top of [RainbowKit](https://www.rainbowkit.com/docs/introduction) and [Wagmi](https://wagmi.sh/), so the dev experience is almost the same.  [See here](/zerokit/rainbow-kit) for converting a RainbowKit app to a ZeroKit app.

You will need your `projectId` here to setup the SDK.

```jsx
import 'zerokit/styles.css';
import { ZeroKitProvider } from "zerokit";
```
```jsx
<ZeroKitProvider projectId="<your-project-id>">
    <YourApp />
</ZeroKitProvider>
```

### Step 4: Connect

In `YourApp`, use the `ConnectButton` component to create a connect button, and use Wagmi to interact with the user's wallet.

```jsx
import { ConnectButton } from "zerokit";
```
```jsx live
<ConnectButton label="Social Wallet" />
```

### Step 5: Build

From here on, you would build your app the same way you build any [Wagmi](https://wagmi.sh/) apps.

Keep in mind that the *signer* you obtain from Wagmi (e.g. through [`useSigner`](https://wagmi.sh/react/hooks/useSigner)) is backed by an account abstraction wallet.  As a result, it can be used with the [ZeroKit SDK](/sdk/intro) to enable powerful AA features.


```jsx
import { useSigner } from "wagmi";
import * as zd from '@zerodevapp/sdk';

function MyComponent() {
  const { data: signer } = useSigner()


  const handleClick = async () => {
    zd.execBatch(signer, [ transaction1, transaction2 ])
  }

  return (
    <Button onClick={handleClick}>
      Batch Mint
    </Button>
  )
}
```

[See here](/category/account-abstraction) for examples of AA features.
