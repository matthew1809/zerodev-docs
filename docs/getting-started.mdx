# Getting Started

### Step 1: Sign up

Fill out the [closed beta signup form](https://docs.google.com/forms/d/e/1FAIpQLSfip6bMIYvd2Z5LDr3IRt-kCEc6x38TybcDe6gk_TRKvT-fMw/viewform) to get a `projectId`.


### Step 2: Installation
<Tabs>
  <TabItem value="npm" label="npm" default>
    <pre>
        <codeblock>npm install --save zerokit</codeblock>
    </pre>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <pre>
        <codeblock>yarn add zerokit</codeblock>
    </pre>
  </TabItem>
</Tabs>

### Step 3: Setup the SDK

ZeroKit is built on top of [RainbowKit](https://www.rainbowkit.com/docs/introduction) and [Wagmi](https://wagmi.sh/), so the dev experience is almost the same.  [See here](/rainbow-kit) for converting a RainbowKit app to a ZeroKit app.

You will need your `projectId` here to setup the SDK.

```jsx
import 'zerokit/styles.css';
import { ZeroKitProvider } from "zerokit";
```
```jsx
<ZeroKitProvider projectId="<your-project-id>">
    <YourApp />
</ZeroKitProvider>
```

### Step 4: Connect

In `YourApp`, use the `ConnectButton` component to create a connect button, and use Wagmi to interact with the user's wallet.

```jsx
import { ConnectButton } from "zerokit";
```
```jsx live
<ConnectButton label="Social Wallet" />
```

### Step 5: Build

From here on, you would build your app the same way you build any [Wagmi](https://wagmi.sh/) apps.

[See here](/examples/sponsored-mint) for a full example.
